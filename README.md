# carrito-compras-mvc-capas
| Proyecto universitario UPC - 202601, Sistemas Operativos | 

## Arquitectura
- Patrón MVC con separación en capas
- Capa controllers, services y repositories
- **Singleton** para la conexión a la base de datos

## Stack
- **LAMP** (Linux, Apache, MySQL, PHP)
    - Linux -> Ubuntu Server en Azure
    - Apache -> Servidor web
    - MySQL -> Base de datos
    - PHP -> Lenguaje Backend

## Instalación
1. Clonar repositorio
2. Copiar config/env.example.php a config/env.php
3. Agregar credenciales de la BD a env.php
